Cultural Notes Generator

A simple Python script that uses OpenAI GPT to turn keywords or transcript into short cultural explanations.

How to Use
1. Clone or download the project folder to your computer.
2. Install the required Python packages:
3. pip install openai python-dotenv
4. Add your OpenAI key to a `.env` file:
OPENAI_API_KEY=your_key_here
5. Run:
python cultural_notes.py

Example Input

kimono, tea ceremony, Japan

Example Output
> The kimono and tea ceremony are traditional symbols of Japanese culture. The kimono reflects grace and heritage, while the tea ceremony emphasizes harmony, respect, and mindfulness.
